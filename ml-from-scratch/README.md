# Machine Learning From Scratch

A growing collection of mini-projects built alongside the *Machine Learning
Specialization* (DeepLearning.AI / Stanford Online, via Coursera). Each
topic in the course gets its own folder here, with a small standalone
project that applies the concept to a dataset that's different from the
one used in the course labs — implemented from scratch wherever the
concept itself is the point (no scikit-learn shortcuts on the core
algorithm).

## Why this repo exists

Course certificates show you finished something. This shows you can build
with what you learned.

## Tech

Python, NumPy, pandas, Matplotlib, Jupyter. Core algorithms (model
functions, cost functions, gradient descent, etc.) are implemented from
scratch to demonstrate understanding, not just library usage.
